# Demo Questions for ASL Translation System

These questions are designed to be answered using words from the WLASL-100 class vocabulary.

## Interview Questions

| Question | Possible Answers (from vocab) |
|----------|------------------------------|
| What do you want to do for work? | work, cook, doctor, secretary |
| Can you tell me about your family? | mother, son, wife, family, cousin |
| When can you start? | now, later, thursday |
| Do you like to work with people? | yes, no, like |
| What did you study in school? | school, study, language, computer |

## Doctor's Office Questions

| Question | Possible Answers (from vocab) |
|----------|------------------------------|
| Do you need help? | yes, no, help |
| Are you hot or cold? | hot, cool, fine |
| Do you want water? | yes, no, want, water |
| Can you walk? | yes, no, walk |
| Did you eat before coming? | yes, no, eat, before |

## Available Vocabulary (100 classes)

The model can recognize the following 100 words:

**Actions:** book, drink, go, walk, help, finish, like, kiss, graduate, study, play, dance, eat, work, meet, give, forget, change, wait, decide, cheat, need, cook, tell

**People/Family:** cousin, mother, woman, man, family, wife, son, doctor, secretary

**Objects:** chair, clothes, candy, table, bed, fish, hat, shirt, bird, apple, corn, cow, pizza, water, letter, jacket, paper, dog, computer, basketball, bowling

**Colors:** black, orange, blue, pink, white, purple, brown, yellow, dark

**Descriptions:** deaf, thin, fine, hot, cool, hearing, tall, wrong, short, full, right, same

**Time:** before, year, now, later, last, time, thursday, birthday

**Food:** candy, fish, apple, corn, pizza, water, drink, eat, cook

**Other:** yes, no, what, who, all, many, thanksgiving, language, color, enjoy, accident, school, city, africa, can, but, want, need

## Tips for Demo

1. Ask clear, simple questions
2. Give the signer time to respond (2-3 seconds per sign)
3. Watch for the translation to appear after 3 signs
4. Common responses like "yes", "no", "help", "want" work well
